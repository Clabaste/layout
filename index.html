Layouting mit CSS – Flex vs Grid

Das flexible Box-Modul, „Flexbox“ abgekürzt, ist ein eindimensionales Layout-Modell, erzeugt also entweder Zeilen oder Spalten. Mit Flexbox lassen sich Elemente gut in einem Container ausrichten und verteilen.

Mit CSS-Grid kann man noch die zweite Dimension ins Layout nehmen und gleichzeitig sowohl Zeilen als auch Spalten erzeugen.

Mittel CSS-Grid kann man ein 2-dimensionales Layout erzeugen, also sowohl Zeilen als auch Spalten des Layouts definieren.

CSS-Grid arbeitet mit einer 12-Raster-Anordnung, bei der der Bildschirm (unsichtbar) in 12 Teile unterteilt ist und Elemente in diese Anordnung passen müssen.

Der Vorteil von CSS Grid gegenüber Flexbox und anderen Layoutmodellen ist seine zweidimensionale Qualität . Es macht auch die Positionierung einfacher, und die Elemente des Containers können so eingestellt werden, dass sie sich überlappen und überlappen.

Prinzip Flex

Flexbox geht von der Größe des Inhalts aus.

Die Breite der Kindelemente richtet sich entweder nach deren Inhalt oder nach einer Breitenangabe. Alternativ besteht die Möglichkeit, einzelne oder alle Kindelemente so auszurichten, dass sie immer die gesamte Breite des Elternelementes ausfüllen

Prinzip Grid

Ein Wert von „1fr“ macht alle Spalten und Zeilen gleich groß und verteilt sie über den zur Verfügung stehenden Platz. Will man eine Spalte doppelt so groß haben wie die anderen, weist man ihr „2fr“ zu. Natürlich können aber auch absolute Einheiten wie Pixel verwendet werden.

Überschneidungen?

Auch wenn "grid" für zweidimensionale Layouts prädestiniert ist, kann man vieles sowohl mit "flex" oder mit "grid" lösen – es ist eben auch ein bisschen Geschmackssache.

Generell kann man sagen Flexbox bietet mehr Flexibilität. Grid hilft bei der Erstellung eines kontrollierteren Layouts – hier wird im Elternelement ein Gitter aus Zeilen und Spalten definiert.

    <main>Content</main>
    <aside>Marginal</aside>
    <footer>Footer</footer>

/*  Elternelement */
.grid-complex-example {
    display: grid;
    gap: 10px;
    grid-template-columns: 60px auto 100px;
    grid-template-rows: 100px auto auto;
    border: 2px dashed darkblue;
    width: 100%;
}

/*  Kindelement: erste Zeile geht über zwei Zeilen */
.grid-complex-example header {
    background-color: cornflowerblue;
    grid-column-start: 2;
    grid-column-end: span 2;
}
.grid-complex-example nav {
    background-color: deepskyblue;
    grid-row: 1/3;
}
.grid-complex-example aside {
    background-color: lightblue;
    grid-column-start: 3;
    grid-row: 2/4;
}
.grid-complex-example footer {
    background-color: aliceblue;
    grid-column: 1/3
}

Grid für komplexe, statische Designs.
